# vowel Figures
 Plots vowels in 2D (using phonR package) and 3D (using scatterplot)
 
## Input
 An Excel file  with formant values and the following format:
  label F1 F2 F3
vocal | f1 | f2| f3
------------ | ------------- | -------------
a | 1000 | 1500| 3000
 
## Output
Figures that can be saved with right click or dragging to Desktop.
